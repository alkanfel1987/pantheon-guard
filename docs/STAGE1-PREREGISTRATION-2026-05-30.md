# Ступень 1 — пре-регистрация held-out пробы слияния regex ∪ semantic

Зарегистрировано 2026-05-30 ДО сбора корпуса (дисциплина §2: pre-registration перед измерением).
Контекст: nonstop-прогон, критерий «done» = Ступень 1 (`NONSTOP-CHARTER.md`).

## Гипотеза (фиксируется до чисел)
Слияние `union = regex ∪ semantic` на кросс-типовом held-out корпусе даёт
**catch Wilson-LB ≥ 50%** при **FP Wilson-UB ≤ 5%**.

Основание: FE-LIVE проба (N=6, `integration-fe-live-results.json`) — union catch
100% [60.97, 100], FP 0%, комплементарность semantic↔regex идеальна. Underpowered
(N мал, один тип). Эта проба масштабирует до кросс-типа + большего N.

## Корпус (manual-curated, НЕ auto-pull)
- **Позитивы N ≥ 30**, кросс-тип, по 5 классам (≈6 на класс):
  маркетинг/продажи · скам · UX dark patterns · health/wellness · политика/false-equivalence.
- Обязаны задеть классы уязвимости, которые новостной корпус пропустил:
  `lobha` (жадность) · `kāma` (желание) · `shame` (стыд) · доверие к авторитету.
- **Негативы N ≥ 30** (цель ≥60 для FP-UB ≤ 5% при 0 FP): нейтральные тексты
  тех же регистров (wire-репортаж, академический explainer, легитимный CTA).
- **Источники verbatim**, не перефраз. Кандидатный источник по идее пользователя:
  callout-комментарии устойчивого меньшинства («это кликбейт / подмена / давят на вину»)
  — как указатель на манипулятивные образцы; финальная метка — адъюдикация пользователя.
- Источники **не пересекаются** с теми, на которых строились детекторы (анти-overfit).

## Разметка (двухосевая, мульти-тег)
Каждый образец: `уязвимость × техника`, мульти-тег primary+secondary
(модель манипуляции 4-слойная, handoff 2026-05-19).
- **Первый проход — Claude** (только кандидат-метки).
- **Адъюдикация — пользователь** (Q1, связующий шаг). Без неё результат = sanity-check, не validation (Я-2).

## Решающее правило (фиксируется ДО чисел)
- **PASS Ступени 1** ⟺ union catch Wilson-LB ≥ 50% И union FP Wilson-UB ≤ 5%.
- **FAIL** = валидное завершение (честный негативный вердикт, как L2-CONCLUSION). Пороги НЕ крутить.
- Без threshold-tuning на этом корпусе (cycle-2 trap). Порог semantic (0.55) фиксирован из FE-пробы.

## Что НЕ входит
- Ступень 2 (2-й аннотатор + Cohen's κ) — нужен второй человек, вне прогона.
- Любое внешнее число/claim из этого корпуса до прохождения адъюдикации.
