## C. Bridges — Pantheon-internal structural observations (NOT documented equivalences)

The model draws structural parallels between Sanskrit vice-taxonomies,
Western persuasion-psychology levers, and cognitive-architecture levers.
**These are Pantheon-internal observations.** They are useful for internal
analysis and catalogue organisation. **They are NOT peer-reviewed
equivalences** in the cross-cultural psychology or comparative-religion
literature, and they must not be cited as such.

### Bridge B1 — ṣaḍ-ripu ↔ Cialdini's principles ↔ persuasion-psychology levers

**Observation (Pantheon-internal):** the six classical Sanskrit vices
(*kāma · krodha · lobha · moha · mada · mātsarya*) cover a substantial part
of the levers later catalogued in Western persuasion psychology (Cialdini's
six principles; the broader moral-emotion taxonomies). E.g.:
- `lobha` ↔ Cialdini scarcity / reciprocity / commitment all engage gain-seeking
- `mada` ↔ ego/status-flattery levers in marketing
- `kāma` ↔ sense-pleasure / comfort-craving in lifestyle advertising

**Status:** **NOT a documented equivalence.** No peer-reviewed cross-
traditional psychology paper maps ṣaḍ-ripu to Cialdini's principles. The
mapping is an internal Pantheon analytical convenience — it lets the Vedic
Manipulation Catalogue use Sanskrit class names without forcing the user to
translate every modern example. **The model does not claim ṣaḍ-ripu and
Cialdini "are the same thing."**

### Bridge B2 — Kleshas ↔ cognitive biases

**Observation (Pantheon-internal):** some Yoga-Sūtra *kleśas* (Patañjali
2.3: *avidyā · asmitā · rāga · dveṣa · abhiniveśa*) parallel modern
cognitive-bias categories — `avidyā` (mis-knowing as background) parallels
the inattention/default-bias surfaced by Condition 3 (Thaler-Sunstein);
`abhiniveśa` (clinging to life / death-fear) parallels loss-aversion
(Kahneman-Tversky).

**Status:** **NOT a documented equivalence.** Patañjali's *kleśas* are an
ontological-soteriological scheme, not a cognitive-bias taxonomy. The
mapping is an internal Pantheon framing only.

### Bridge B3 — Mahā-vrata violations ↔ manipulation taxonomy

**Observation (Pantheon-internal):** Pantheon's *mahā-vrata* (the five
yamas of Yoga-Sūtra 2.30 — *satya · ahiṃsā · asteya · brahmacarya/indriya-
nigraha · aparigraha/shaucha*) are used in this project as the ethical
filter set under which manipulation is *evaluated*. Each manipulation type
can be routed to one or more mahā-vrata it violates (e.g. fabricated-
authority violates *satya*; coercive marketing violates *ahiṃsā*).

**Status:** **NOT a documented equivalence.** The mahā-vrata are an
*ethical* canon from Patañjali, not a manipulation-classification scheme.
Their use as such is a **Pantheon architectural choice** — the routing
table is project-internal (Vedic Manipulation Catalogue / Foundation).

### Bridge B4 — Cross-source Layer-3 unification

**Observation (Pantheon-internal):** the Layer-3 technique vocabulary
unifies items drawn from at least four separate literatures — Cialdini's
persuasion principles (1984), Brignull's dark-patterns catalogue (2010+),
Stalnaker/Karttunen's pragmatic-presupposition theory (1970s),
Thaler-Sunstein's choice architecture (2008) — into a single open-ended
list under the model's Layer 3.

**Status:** **Pantheon-synthesized unification.** No single source covers
all the technique families used in the validation corpus. The unification
is the model's organisational contribution; it is **not** a claim that
these literatures are "really" one literature.
